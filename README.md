# Sistema POS Profesional - PHP

Sistema de Punto de Venta (POS) moderno, responsivo y profesional desarrollado en PHP.

## Características
- **Productos**: Gestión completa, categorías y ubicaciones.
- **Ventas**: Punto de venta multi-ventana, devoluciones y facturación electrónica (SRI).
- **Inventario**: Kardex, ajustes, órdenes de compra y traslados.
- **Caja**: Gestión de turnos, cierres diarios y reportes.
- **Configuración**: Perfil de empresa y gestión de impuestos.

## Requisitos
- PHP 7.4+
- MariaDB / MySQL
- Servidor Web (Apache/Nginx)
